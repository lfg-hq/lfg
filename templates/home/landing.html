Get Started Locally

Set up LFG on your machine in just a few minutes

1

Clone the Repository

git clone https://github.com/lfg-hq/lfg.git

Get the latest version of LFG from our GitHub repository

2

Create Virtual Environment

cd lfg && python -m venv venv && source venv/bin/activate

Set up an isolated Python environment for the project

3

Install Dependencies

pip install -r requirements.txt

Install all required Python packages and dependencies

4

Set Up Environment

cp .env.example .env

Configure your environment variables and add your API keys

5

Run Database Migrations

python manage.py migrate

Set up the database schema and initial data

6

Start the Development Server

uvicorn LFG.asgi:application --host 0.0.0.0 --port 8000 --workers 2

Launch LFG locally and access it at http://localhost:8000
